Platform Description

Codebase Analyzer is a CLI-launched web application for AI-assisted code audits of a target repository. It runs an Express + Socket.IO backend and a React dashboard to generate, review, and iteratively refine per-domain documentation, diagrams, requirements, bugs/security findings, and refactoring/testing recommendations. All analysis artifacts, task state, logs, and chat history are persisted into the target project's .code-analysis/ directory.
